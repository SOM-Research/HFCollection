The Community database of Hugging Face Hub

What is HFCollection?

HFCollection offers an offline up-to-date relational database built from the data available at the Hugging Face Hub, thus providing queriable data about the repositories hosted in the Hub.

Thanks to HFCollection, developers can quickly get any kind of information about the projects hosted on Hugging Face Hub.

 

Why Hugging Face?

Hugging Face has become one of the most representative AI Open Source hosting platform.

HFCollection offers a convenient access to query Hugging Face Hub Data
